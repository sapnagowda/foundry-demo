# 02 — Application Insights (Tracing Enabled)

## Goal
Attach Application Insights to the Foundry project so agent runs appear in both Foundry tracing and App Insights “Agents view”.

## You will end with
- Application Insights resource connected to the Foundry project
- Tracing enabled and ready for OpenTelemetry correlation

- Completed `../01-foundry-project-and-model-deployment`

## Proof (checklist)
- [ ] Foundry project is linked to App Insights
- [ ] You can locate the resource in Azure and open it
- [ ] (Later) agent runs appear in App Insights once instrumentation is added
